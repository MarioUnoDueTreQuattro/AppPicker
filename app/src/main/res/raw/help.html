App Picker


This app try to alleviate problems on some Android versions:

  •  No automatic default app if you don't have the "Just once" option.

  • No need to press "Just once" if you have the "Just once" option.

How:
  •  In the app chooser choose "App Picker".
  • If  you have the "Just once" option, check it to set "App Picker" as default app.

Now you can choose.
